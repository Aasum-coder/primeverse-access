## Bugs Fixed

1. **Traffic Sources showed global data** — Every IB saw the same numbers. Fixed by adding `Cache-Control: no-store` and `force-dynamic` to the API route.
2. **Traffic Sources ignored time filter** — Modal always showed 30 days. Fixed by passing `period` prop and query param to the API.
3. **Disapproved leads stayed in list** — Added `rejected` boolean column, set it on disapprove, filter it out in UI.
4. **UID submission failed ("Something went wrong")** — Root cause: `.select('id').single()` after INSERT failed because anon role has no SELECT RLS policy. Fixed by generating UUID client-side.
5. **PWA opened in Safari, not standalone** — Missing PNG icon files referenced by manifest.json. Generated all required sizes.
6. **No form validation on UID submission** — Empty fields were submittable. Added required-field and email-format validation with inline errors.
7. **Avatar crop broken on mobile** — Touch events existed but lacked `preventDefault()`, `touch-action: none`, and pinch-to-zoom. Replaced with full canvas-based editor modal.

## Features Built

1. **Telegram Bot Integration** — Full webhook system for SYSTM8NotifyBot. IBs click deep link in Profile tab → bot saves their `chat_id` → they receive Telegram notifications when leads submit UIDs.
2. **Avatar Editor Modal** — Canvas-based image editor with drag-to-pan, pinch-to-zoom, zoom slider, brightness slider, circular preview, and confirm/cancel buttons. Works on iPhone Safari, Android Chrome, and desktop.
3. **UID Form Pre-fill** — Registration data (name + email) from Step 1 automatically fills the UID form in Step 5 via localStorage.
4. **Prebuild Icon Generation** — `scripts/generate-icons.mjs` runs during Vercel builds to download the real 1Move logo and resize to all PWA icon sizes.
5. **Landing Page Copy Update** — "How It Works" section updated to "Five Steps to Free Access" with new messaging in all 9 languages.
